# Blackboard Implementation

Project to implement the algorithm with blackboard

![Blackboard](https://imgur.com/7PTReP2)

## Instructions

You need to have `sbt` installed to run these steps

#### Building 
    
    sbt compile
    
#### Test

    sbt test

### Run

    sbt run

#### Package (Fat JAR)

    sbt assembly
    
#### Running (Fat JAR)

    java -jar target/scala-*/blackboard*.jar
