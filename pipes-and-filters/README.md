# Pipes and Filters Implementation

Project to implement the alogirthm with pipes and filters

## Instructions

You need to have `sbt` installed to run these steps

#### Building 
    
    sbt compile
    
#### Test

    sbt test

#### Package (Fat JAR)

    sbt assembly
    
#### Running (Fat JAR)

    java -jar target/scala-*/pipes-and-filters-assembly*.jar
